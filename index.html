App Failed to Load / 应用无法加载

If you are seeing this, you likely opened index.html directly using a file manager. Modern web apps require a web server to run securely.

如果您看到此页面，说明您可能直接双击打开了文件。 请使用以下免费方式运行：

Option 1: Local (本地运行)

npm install
npm run dev

Option 2: Deploy (部署上线)

Push code to GitHub and use Vercel or GitHub Pages.
